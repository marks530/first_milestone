What is an ICC Profile?

An ICC Profile controls the color management of your printer. A color profile is designed to get the best quality prints from your printer – it changes the preset colours of your printer ink.

Colour Charts

We use the latest colour management software called iPublish and the latest Spectrophotometer, the iPro 3, both from x-Rite

Colour Calibration

We create custom made icc color profiles for your printer – all you need to do is print the colour chart on printer using the instructions on our website – do this using the ink type and paper type you want an ICC profile
